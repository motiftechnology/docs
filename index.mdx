## What is ChatAfrika?

ChatAfrika is a comprehensive multi-tenant chat and customer support platform built with modern technologies. It provides real-time messaging, AI-powered bots, webhooks, and a complete SDK for integration.

## Key Features

- **Multi-tenant Architecture**: Isolated tenants with their own data and configuration
- **Real-time Messaging**: WebSocket-based instant messaging with read receipts
- **AI Bot Integration**: OpenAI-powered conversational bots with handover to agents
- **Support Routing**: Intelligent conversation routing to departments and agents
- **Webhooks**: Event-driven webhooks for external integrations
- **Presence & Typing Indicators**: Real-time user presence and typing status
- **Analytics**: Bot analytics, feedback collection, and metrics

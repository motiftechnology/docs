## Prerequisites

Before you begin, you'll need:

- A ChaterAfrika account (contact [support@chaterafrika.com](mailto:support@chaterafrika.com) for access)
- Your API credentials (API URL and SDK token)
- A modern web browser or Node.js environment

## JavaScript SDK

The easiest way to integrate ChaterAfrika into your application is using our JavaScript SDK.

### Installation

```bash
npm install @chatafrika/sdk
```

### Basic Setup

```typescript
import { ChatAfrika } from '@chatafrika/sdk';

// Initialize the SDK with your credentials
const client = new ChatAfrika({
  wsUrl: 'wss://ws.dev.chaterafrika.com', // Your WebSocket endpoint
  token: 'your-sdk-token' // Get this from your dashboard
});

// Connect to WebSocket
await client.connect();
```

### Join a Conversation

```typescript
// Join a conversation to receive messages
await client.conversations.join('conversation-id');

// Get the conversation instance
const conversation = client.conversations.get('conversation-id');

console.log('Joined conversation:', conversation.id);
```

### Send a Message

<Warning>
**Messages are sent exclusively over WebSockets.**

This guarantees ordering, real-time delivery, typing indicators, presence, and receipts.

Use `client.conversations.get(conversationId).sendMessage(content)` to send messages.

REST APIs are used for history, analytics, and internal automation only.
</Warning>

```typescript
// Get the conversation and send a message
const conversation = client.conversations.get('conversation-id');
conversation.sendMessage('Hello, world!');

// Or in one line:
client.conversations.get('conversation-id').sendMessage('Hello, world!');
```

### Receive Messages

```typescript
// Get the conversation and listen for incoming messages
const conversation = client.conversations.get('conversation-id');

conversation.on('message', (message) => {
  console.log('New message:', message.content);
  console.log('From:', message.senderId);
  // Update your UI here
});
```

### Typing Indicators

```typescript
const conversation = client.conversations.get('conversation-id');

// Start typing indicator
conversation.startTyping();

// Stop typing indicator
conversation.stopTyping();

// Listen for typing events from others
conversation.on('typing', (event) => {
  if (event.state === 'start') {
    console.log(`${event.userId} is typing...`);
  } else {
    console.log(`${event.userId} stopped typing`);
  }
});
```

### Presence

```typescript
// Listen for presence updates at the SDK level
client.on('presence', (event) => {
  console.log(`${event.userId} is ${event.state}`);
  // event.state is 'online' or 'offline'
});
```

### Complete Example

```typescript
import { ChatAfrika } from '@chatafrika/sdk';

async function initializeChat() {
  // Initialize SDK
  const client = new ChatAfrika({
    wsUrl: 'wss://ws.dev.chaterafrika.com',
    token: 'your-sdk-token',
    debug: true
  });

  // Handle connection events
  client.on('connected', () => console.log('Connected!'));
  client.on('disconnected', () => console.log('Disconnected'));
  client.on('error', (err) => console.error('Error:', err));

  // Connect to WebSocket
  await client.connect();
  console.log('Connected to ChaterAfrika');

  // Join conversation
  const conversationId = 'your-conversation-id';
  await client.conversations.join(conversationId);

  // Get conversation instance
  const conversation = client.conversations.get(conversationId);

  // Listen for messages
  conversation.on('message', (message) => {
    console.log('Received:', message.content);
    // Display message in your UI
  });

  // Listen for typing
  conversation.on('typing', (event) => {
    if (event.state === 'start') {
      console.log(`${event.userId} is typing...`);
    }
  });

  // Listen for receipts
  conversation.on('receipt', (event) => {
    console.log(`Message ${event.messageId}: ${event.state}`);
  });

  // Send a message
  conversation.sendMessage('Hello from ChaterAfrika!');

  return client;
}

// Initialize
initializeChat().catch(console.error);
```

## REST API

You can interact with ChaterAfrika via REST API for reading data and managing resources.

### Authentication

First, authenticate to get your access token:

```bash
curl -X POST https://api.chaterafrika.com/api/v1/auth/login \
  -H "Content-Type: application/json" \
  -d '{
    "email": "your-email@example.com",
    "password": "your-password"
  }'
```

Response:
```json
{
  "access_token": "eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9...",
  "refresh_token": "eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9...",
  "user": { ... }
}
```

### Get SDK Token

Exchange your access token for an SDK token:

```bash
curl -X POST https://api.chaterafrika.com/api/v1/auth/sdk-token \
  -H "Authorization: Bearer YOUR_ACCESS_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{
    "app_id": "your-app-id"
  }'
```

Response:
```json
{
  "sdk_token": "eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9..."
}
```

### Make API Requests

Use the access token in subsequent requests:

```bash
# Get conversations
curl https://api.chaterafrika.com/api/v1/conversations \
  -H "Authorization: Bearer YOUR_ACCESS_TOKEN"

# Get messages for a conversation
curl https://api.chaterafrika.com/api/v1/conversations/{conversationId}/messages \
  -H "Authorization: Bearer YOUR_ACCESS_TOKEN"
```

<Note>
**REST API is for reading data only.** Use the SDK for sending messages.
</Note>

## WebSocket Connection

For real-time features, connect to the WebSocket service:

```typescript
// Using the SDK (recommended)
await client.connect();

// Or manually
const ws = new WebSocket('wss://ws.dev.chaterafrika.com/ws', {
  headers: {
    'Authorization': `Bearer ${sdkToken}`
  }
});

ws.onopen = () => {
  // Join a conversation
  ws.send(JSON.stringify({
    type: 'join',
    conversation_id: 'conversation-id'
  }));
};

ws.onmessage = (event) => {
  const message = JSON.parse(event.data);
  console.log('Received:', message);
};
```

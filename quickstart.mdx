## Prerequisites

Before you begin, you'll need:

- A ChaterAfrika account (contact [support@chaterafrika.com](mailto:support@chaterafrika.com) for access)
- Your API credentials (API URL and SDK token)
- A modern web browser or Node.js environment

## JavaScript SDK

The easiest way to integrate ChaterAfrika into your application is using our JavaScript SDK.

### Installation

```bash
npm install @chatafrika/sdk
```

### Basic Setup

```typescript
import { ChatAfrika } from '@chatafrika/sdk';

// Initialize the SDK with your credentials
const client = new ChatAfrika({
  apiUrl: 'https://api.chaterafrika.com', // Your API endpoint
  wsUrl: 'wss://realtime.chaterafrika.com', // Your WebSocket endpoint
  sdkToken: 'your_sdk_token' // Get this from your dashboard
});

// Connect to realtime
await client.connect();
```

### Create a Conversation

```typescript
// Create a new conversation
const conversation = await client.conversations.create({
  type: 'DIRECT',
  members: ['user-id-1', 'user-id-2']
});

console.log('Conversation created:', conversation.id);
```

### Send a Message

```typescript
// Send a message
const message = await client.messages.send({
  conversationId: conversation.id,
  content: 'Hello, world!'
});

console.log('Message sent:', message.id);
```

### Receive Messages

```typescript
// Join a conversation to receive messages
await client.conversations.join(conversation.id);

// Listen for incoming messages
client.messages.on('message', (message) => {
  console.log('New message received:', message);
  // Update your UI here
});
```

### Presence and Typing Indicators

```typescript
// Update presence status
await client.presence.update('online');

// Send typing indicator
await client.typing.start(conversation.id);

// Stop typing indicator
await client.typing.stop(conversation.id);

// Listen for presence updates
client.presence.on('update', (update) => {
  console.log('User presence changed:', update);
});
```

### Complete Example

```typescript
import { ChatAfrika } from '@chatafrika/sdk';

async function initializeChat() {
  // Initialize SDK
  const client = new ChatAfrika({
    apiUrl: 'https://api.chaterafrika.com',
    wsUrl: 'wss://realtime.chaterafrika.com',
    sdkToken: 'your_sdk_token'
  });

  // Connect to realtime
  await client.connect();
  console.log('Connected to ChaterAfrika');

  // Create or get conversation
  const conversation = await client.conversations.create({
    type: 'DIRECT',
    members: ['user-id-1', 'user-id-2']
  });

  // Join conversation
  await client.conversations.join(conversation.id);

  // Listen for messages
  client.messages.on('message', (message) => {
    console.log('Received:', message.content);
    // Display message in your UI
  });

  // Send a message
  await client.messages.send({
    conversationId: conversation.id,
    content: 'Hello from ChaterAfrika!'
  });

  return client;
}

// Initialize
initializeChat().catch(console.error);
```

## REST API

You can also interact with ChaterAfrika directly via REST API.

### Authentication

First, authenticate to get your access token:

```bash
curl -X POST https://api.chaterafrika.com/api/v1/auth/login \
  -H "Content-Type: application/json" \
  -d '{
    "email": "your-email@example.com",
    "password": "your-password"
  }'
```

Response:
```json
{
  "access_token": "eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9...",
  "refresh_token": "eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9...",
  "user": { ... }
}
```

### Make API Requests

Use the access token in subsequent requests:

```bash
# Get conversations
curl https://api.chaterafrika.com/api/v1/conversations \
  -H "Authorization: Bearer YOUR_ACCESS_TOKEN"

# Send a message
curl -X POST https://api.chaterafrika.com/api/v1/messages \
  -H "Authorization: Bearer YOUR_ACCESS_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{
    "conversation_id": "conversation-uuid",
    "content": "Hello, world!"
  }'
```

## WebSocket Connection

For real-time features, connect to the WebSocket service:

```typescript
// Using the SDK (recommended)
await client.connect();

// Or manually
const ws = new WebSocket('wss://realtime.chaterafrika.com/ws', {
  headers: {
    'Authorization': `Bearer ${sdkToken}`
  }
});

ws.onopen = () => {
  // Join a conversation
  ws.send(JSON.stringify({
    type: 'join',
    conversation_id: 'conversation-id'
  }));
};

ws.onmessage = (event) => {
  const message = JSON.parse(event.data);
  console.log('Received:', message);
};
```
